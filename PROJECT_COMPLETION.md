## 🚀 How to Use

### Quick Start (Pick One)

**Fastest (GUI with splash):**
```bash
Double-click: launcher.bat
```

**Direct Launch:**
```bash
Double-click: run.bat
```

**Standalone EXE:**
```bash
Double-click: build_exe.bat        (wait 2-3 min)
Then: dist/OpenModelica Simulator.exe
```

### Running a Simulation

1. **Launcher shows** → Click "LAUNCH APPLICATION"
2. **GUI appears** → Enter desired time range
3. **Click START** → Simulation begins
4. **Watch progress** → Bar fills, timer counts
5. **See results** → Completion message appears
6. **Check output** → Results in model/ folder
